Designed and Developed by : Farrukh Shahzad and Dr. Atif Zaman Khan

Programming Language : Java (Sun JDK)

Compatible Browser : Any Java compatible browser: IE 3.x, Navigator 3.x, etc. This version runs in a separate window, outside of the browser.

Description : An interactive graphical Applet for the analysis of single-machine infinite-bus (SMIB) power system transient stability using Equal area criteria (EAC). It plots interactive power angle curves with on-line parameter adjustment showing accelerating/ decelerating areas, operating and clearing angles, stability margins, etc., compute critical clearing angle (CCA) and other stability parameters. This applet is a portion of complete software package PSS_EAC, which is a DOS version developed in Pascal (with interactive GUI, full menus, on-line help and solved demo examples) . This software is believed to be the first educational use of Java applet for power system faculty & students on the internet. This software is a supplement to the following IEEE Transaction papers.

[1] F. Shahzad, Atif Z. Khan, "An application of Java for power system stability," Power Engineering Letters, IEEE Power Engineering Review, pp. 45-46, Vol. 18, No. 4, April 1998.

[2] Atif Z. Khan, F. Shahzad, "A PC based software package for the equal area criterion of power system transient stability," IEEE Trans. Power Systems, Vol. 13, No. 1, pp. 21-26, Feb. 1998

[3] Atif Z. Khan, F. Shahzad, "Step-Change in generator mechanical power: Comprehensive analysis," IEEE Trans. Power Systems, Vol. 12, No. 2, pp. 869-874, May 1997.
