GPS User Interface

This version of Applet is not supported in this browser.

Designed and Developed by : Farrukh Shahzad

Programming Language : Java (Sun JDK)

Compatible Browser : Any Java 1.1 enabled browser: IE 4.x, Navigator 4.x, Hot Java, etc.

Description : An Interactive Applet to plot USA/World Map. User can scroll around the map with zoom in and zoom out Features. User can also select any State/Country, with added features of displaying major US cities and US highways. This applet also print name of State/Country, name of cities, Longitude/Latitude position. It also computes shortest distance in miles between any two locations on the map.

Applications : This applet can provide an interactive user interface for Global Positioning System (GPS), Position Tracking, Remote Monitoring, Cargo Montoring, Breakdown Notification.

Other Features :
  • Real time loading of data, i.e. instant view of applet (No wait for data loading).
  • Use of new ScrollPane class.
  • Use of new efficient methods of file I/O of java.io.* package.
  • Use of new compressed file I/O package java.util.zip.* for faster downloading of map data.
