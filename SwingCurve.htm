Programming Language : Java (Sun JDK)

Compatible Browser : Any Java compatible browser: IE 3.x, Navigator 3.x, etc.

Description : An interactive graphical Applet for the analysis of single-machine infinite-bus (SMIB) power system transient stability using Equal area criteria (EAC). It plots interactive power angle curves with on-line parameter adjustment showing accelerating/ decelerating areas, operating and clearing angles, stability margins, etc., compute critical clearing angle (CCA) and other stability parameters. This applet is a portion of complete software package PSS_EAC, which is a DOS version developed in Pascal. This version runs within the browser.
